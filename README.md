# Testiranje1

This is a simple Flask application that prints the visitor IP address and detects the country using [ipapi.co](https://ipapi.co/).

## Running the app

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the server:
   ```bash
   python app.py
   ```
3. Open a browser at `http://localhost:5000`.

The application will display a greeting with your IP address and country.
